# AI City Agent

AI-агент, который помогает туристу построить идеальный пеший маршрут в любой точке мира.
Он не просто ищет места, а решает задачу коммивояжера, экономя время и силы пользователя.

## Как это работает?
Проект совмещает два трека:
- **Трек A (Внешние API):** Агент полностью автономен. 
Он сам принимает решение вызвать геокодер для перевода адреса в координаты, 
а затем парсит OpenStreetMap, чтобы найти реальные здания и заведения поблизости.

- **Трек D (Оптимизация):** Найденные локации прогоняются через математический алгоритм.
Агент рассчитывает случайный маршрут (Baseline) и оптимизированный (Greedy Algorithm через гаверсинус),
выдавая пользователю чистую выгоду в сэкономленных километрах.

## Инструменты агента
1. `geocode_place` — перевод текстового адреса в GPS (Nominatim API).
2. `get_nearby_places` — поиск реальных заведений (Overpass API / OpenStreetMap).
3. `optimize_route` — локальный Python-модуль для математической оптимизации пути.


## Запуск

1. Клонируйте репозиторий.
2. Создайте файл `.env` и добавьте туда свой ключ: `GROQ_API_KEY=ваш_ключ`
3. Запустите агента через uv:
```bash
uv run main.py
```
